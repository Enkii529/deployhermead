# jasonead.com – Experimental Site Build

This is a **static HTML/CSS/JS** version of the future jasonead.com website built inside the OpenClaw workspace.

## How to Preview

1. If you have a simple HTTP server (e.g., `python -m http.server` or `pnpm dlx http-server`), run it from the project root:
   ```bash
   cd /home/openclaw/openclaw-command-center/projects/jasonead-site-rebuild
   python -m http.server 8000
   ```
2. Open your browser and navigate to `http://localhost:8000`. The site will load with the dark‑mode default.
3. The site is fully responsive with a minimal but complete visual flow.

## What’s Included

- `index.html` – Main page with all required sections (hero, about, projects, AI & automation, crypto, creative, contact, footer).
- `assets/css/style.css` – Basic dark‑mode styling, responsive layout.
- `assets/js/main.js` – Placeholder for future functionality.
- `README.md` – This instruction.
